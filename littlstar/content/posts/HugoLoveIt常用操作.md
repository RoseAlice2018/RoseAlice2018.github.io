#### 新建一篇文章
- hugo new posts/new.md 
注意新md文件最好放在posts目录下
- hugo
生成静态文件
- hugo serve（option）
在本地预览
- cd public && git add .&& git commit -m "" &&git push origin main
上传

#### 文章添加tag
- 前沿配置添加
```
tags: ["tags"]
```
